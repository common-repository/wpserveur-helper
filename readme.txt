Outils et liens utiles WPServeur.

== Description ==
Installez et activez WPServeur Helper pour obtenir des conseils de gestion de votre WordPress, ainsi que de nombreux liens utiles pour votre gestion chez WPServeur !

Vous n'êtes pas encore client WPServeur ? Qu'attendez-vous ? [Regardez les offres !](https://www.wpserveur.net/?refwps=206 "Offres Hébergement WordPress WPServeur") :)

== Installation ==
Installez ce plugin en le téléchargeant depuis votre administration WordPress et allez dans le menu de gauche \"WPServeur Helper\" pour accéder à la page du plugin.
